# Token Mint Project

## Goal
Mint a custom SPL Token on Solana Devnet using TypeScript.

---

## Steps
1. **Initialize Mint**
   - File: `spl_init.ts`
   - Created a new token mint on Devnet.
2. **Mint Tokens**
   - File: `spl_mint.ts`
   - Created ATA (Associated Token Account) for my wallet and minted 8.444449 tokens.
